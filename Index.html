Raspberry-pi Plex Server

Hardware

We start with the latest Raspberry-pi model. Your choice of 4GB or 8GB of ram. Inclose it in our own customer case, with easy access to every available port. The case also has an internal 3.5' hard drive available in many sizes to fit your needs.

Operating System

We use a purpose build Linux distro called Plexbian. We start with the current stable release of Debian and clean out everything we don't need. This leaves you with a very stable and very lightweight OS. This allows more system resources to be available for Plex.

User Interface

While you can choose to use the CLI from the boot screen, the default option brings you to a custom desktop based around the Plex interface. A simple toolbar gives you access to system management tools, leaving the rest of the screen for the Plex interface.
